# Портфолио — Temur Islamov

Личное портфолио, построенное на **React + Vite** и **Bootstrap 5**.

## Стек

- React + Vite
- Bootstrap 5
- SCSS
- EmailJS (форма обратной связи без бэкенда)
- Поддержка трёх языков: RU / EN / UZ

## Запуск локально

```bash
git clone https://github.com/qukssavage/portfolio
cd portfolio
npm install
npm run dev
```
