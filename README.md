# Портфолио — Temur Islamov

Личное портфолио **Темура Исламова** — мобильного разработчика из Ташкента.

Содержит разделы:
- **Обо мне** — краткое описание, интересы
- **Навыки** — стек технологий
- **Проекты** — мобильные приложения, веб-сайты, Telegram-боты
- **Ссылки** — контакты и соцсети

## Стек

- React + Vite
- SCSS (без Bootstrap — кастомные стили)
- EmailJS
- Поддержка трёх языков: RU / EN / UZ

## Запуск локально

```bash
git clone https://github.com/qukssavage/portfolio
cd portfolio
npm install
npm run dev
```
