# Портфолио — Temur Islamov

Личное портфолио **Темура Исламова** — fullstack разработчика из Ташкента.

Содержит разделы:
- **Обо мне** — краткое описание, интересы
- **Навыки** — стек технологий (языки, фреймворки, инструменты)
- **Проекты** — веб-приложения, мобильные приложения, Telegram-боты
- **Контакты** — форма обратной связи и ссылки

## Стек

- React + Vite
- SCSS (кастомные стили без Bootstrap)
- EmailJS
- Поддержка трёх языков: RU / EN / UZ

## Технологии автора

- **Веб:** React, JavaScript, HTML/CSS/SCSS
- **Мобильные:** React Native + Expo, Flutter/Dart
- **Бэкенд:** Python, FastAPI, PostgreSQL, Redis
- **Боты:** aiogram, python-telegram-bot, Anthropic API
- **Деплой:** Railway, Render.com, Supabase, GitHub Pages

## Запуск локально

```bash
git clone https://github.com/qukssavage/portfolio
cd portfolio
npm install
npm run dev
```
